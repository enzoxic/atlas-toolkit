Personalization

Turn on dark mode systemwide

Change the color mode that appears in Windows and your apps

Use the legacy right-click menu

Use the legacy right-click menu over the modern right-click menu

Hide the Recycle Bin on desktop

Hide the Recycle Bin shortcut and show it under This PC in Explorer

Use Legacy Balloon Notifications

Render taskbar notifications as balloons instead of toast notifications

Clear recent background history

Clear recent images used as wallpapers. This keeps default wallpapers.

System

Disable Prefetch and Superfetch

Free up CPU used to index routinely accessed data, this disables memory compression

Disable Background Apps

Stop closed apps running in the background, this disables notifications for some apps

Turn off screen edge swipe

Disabling screen edge swipe fixes the taskbar pop-up when the device is in fullscreen

Turn off multiplane overlay (MPO)

MPO reduces CPU and GPU overhead but may cause stuttering issues on the desktop

Sync Windows clock to UEFI clock

Sync the Windows time to the time set in UEFI instead of syncing to Windows servers

Clear all cached icons and images

Clear all icons and images from the system icon-cache directory. A restart is required.

Privacy & security

Turn off User Account Control

Disabling User Account Control allows apps to automatically elevate system privileges

Enable installation of unsigned drivers

Allow the installation of unsigned drivers, unsigned drivers may contain harmful content

Disable Spectre and Meltdown mitigations

Bypass software patches for Spectre and Meltdown, this should not be disabled for AMD

Disable Windows Defender and related services

Free up CPU used for Windows Defender services. This will keep Windows Firewall enabled.

Enable Transactional Synchronization Extensions (TSX)

Enable TSX on Intel® processors. TSX increases performance but has security vulnerabilities.

Performance

Turn on optimizations for windowed games

Reduce latency by using the new flip presentation model in windowed DirectX® 10 & 11 games

Turn on Fullscreen Optimizations systemwide

Enabling this setting may increase performance and reduce latency for fullscreen apps

Turn on hardware-accelerated GPU scheduling

Reduce latency by allowing the GPU to handle processing scheduling instead of through the CPU

Disable USB selective suspend and USB sleeping

Reduce I/O latency by preventing connected USB ports and devices from going into sleep states

Disable device analytics reporting

Stop services from reporting network usage, battery health, and device utilization to free up CPU

Turn off Task Manager monitoring

Stop services from monitoring network, graphics, and app usage in Task Manager to free up CPU
